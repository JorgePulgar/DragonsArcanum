# The Adventure Vault: Homepage SEO & Component Standards

## 1. Homepage SEO Hierarchy
* **Hero H1**: "Choose your next D&D Book" (Targeting high-intent "Buy" keywords).
* **Hero H2**: "The Definitive Archive for Dungeons and Dragons Book Comparisons & Reviews."
* **Cluster 1**: "Side-by-Side D&D Book Comparisons" (Heading level H3).
* **Cluster 2**: "In-Depth D&D Book Critiques" (Heading level H3).
* **Cluster 3**: "D&D Release Tracker 2026" (Heading level H3).
* **Mini-Post Guide**: "Starting Book" (Heading level H3).
* **About Section**: "The Dragon's Story" (Heading level H3).

## 2. Cluster Layout Rules
* Each cluster must include 3 cards.
* On Mobile: column. On Desktop: row.
* **Call to Action (CTA)**: Every cluster must end with a standard `bg-copper-600` button linking to the respective Hub (e.g., "See all Comparisons").

## 3. "Starting Book" Mini-Post Guide (Stat-Block Style)
This section must strictly follow the "Copper Dragon Stat-Block" visual style:
* **Background**: `bg-[#fdf9f0]` (Parchment).
* **Border Top**: `border-t-8 border-[#c68a3a]` (Copper-500).
* **Dividers**: Use a horizontal rule with a linear gradient `bg-gradient-to-r from-transparent via-[#c68a3a] to-transparent`.
* **Quote Box**: Italicized text inside a `bg-[#f9f3e6]` block with a left-hand `border-l-4 border-[#c68a3a]`.
* **Content Hierarchy**:
    1. Title: "Find Your Starting Point" (Serif font, `text-copper-950`).
    2. Guide Text: Mockup text explaining how to choose.
    3. The Path: Bulleted links to specific internal categories.
    4. FAQ: Integrated 3-item accordion for Schema.
